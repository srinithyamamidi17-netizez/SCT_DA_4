# Marketing Campaign EDA

## Objective
Perform Exploratory Data Analysis (EDA) on a marketing campaign dataset to identify customer behavior, purchasing patterns, and recommend marketing budget allocation based on ROI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Data Cleaning
- Loaded the dataset using Pandas.
- Handled missing values in the Income column using the median.
- Checked and removed duplicate records.
- Converted the customer date column to datetime format.

## Exploratory Data Analysis
Created visualizations for:
- Education Distribution
- Marital Status Distribution
- Income Distribution
- Customer Response
- Product Spending
- Purchase Channels
- Correlation Heatmap

## Key Findings
- Most customers have a Graduation education level.
- Wine products generated the highest spending.
- Purchase behavior varies across Web, Store, and Catalog channels.
- Customer income influences purchasing behavior.

## Recommendations
- Increase marketing investment in the highest-performing purchase channel.
- Promote the highest-selling product categories.
- Target high-income customers with personalized campaigns.
